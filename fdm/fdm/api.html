struct mesh : public flecsi::topo::specialization<flecsi::topo::narray, mesh>

A node-centered (vertices) Cartesian mesh specialization.

Public Types

enum axis

Mesh axes.

The axis identifies a Cartesian coordinate axis on the mesh.

Values:

enumerator x_axis

X-coordinate axis.

enumerator y_axis

Y-coordinate axis.

enum boundary

Boundary.

Identifies the low or high boundary along a particular axis.

Values:

enumerator low

Low axis boundary.

enumerator high

High axis boundary.

enum domain

Mesh domains.

The domain identifies the supported iteration spaces on the mesh.

Values:

enumerator interior

This domain includes the DOFs of the problem.

It does not include the dirichlet boundary vertices.

enumerator logical

This domain includes the “logical” vertices, i.e., those that exist in a raw partitioning of the domain into non-intersecting sets of vertices.

It does not include ghost vertices.

enumerator all

This domain includes all local vertices, including ghost cells.

enumerator global

This domain includes all global cells.

template<axis A, domain DM = interior, bool R = false>
inline auto vertices() const

Return a range over the given axis and domain.

The range can be used to iterate over the cells in the given domain, e.g.: 

for(auto j: m.vertices<mesh::y_axis, mesh::interior>()) {
  for(auto i: m.vertices<mesh::x_axis, mesh::interior>()) {
    u[j][i] = 1.0;
  } // for
} // for

Template Parameters:

  • A – The mesh axis.

  • DM – The mesh domain (excluding global).
